# Installation

## Prerequisites

- Git
- Python 3.11+
- Optional: LaTeX distribution for manuscript builds

## Clone

```bash
git clone git@github.com:alexanderholman/mp1-7-defect-campaign-axis-complexity.git
cd mp1-7-defect-campaign-axis-complexity
```

## Update

```bash
git pull --ff-only
```

## Remove

Delete the cloned directory when no longer needed.
